##Запуск проекта
---
```bash
   git clone <ссылка на репозиторий>
   cd web_lab_1/nginx_lab
```   
	Запустить контейнеры:
```bash
docker-compose up -d --build
```
	Перейти по ссылке в адресной строке вашего проекта:
	```http://localhost:3821```
